# ficha-sarah

Site estático em HTML para publicar no GitHub Pages.

## Arquivos principais
- `index.html`: página principal publicada.
- `remixed-782c0b86.html`: versão original mantida como backup.

## Como publicar no GitHub
1. Faça push para o branch `main`.
2. No GitHub, abra **Settings → Pages** e confirme **Source: GitHub Actions**.
3. Aguarde a workflow `Deploy static HTML to GitHub Pages` finalizar.
4. O site será publicado na URL de Pages do repositório.

## Desenvolvimento local
Abra o `index.html` direto no navegador.
